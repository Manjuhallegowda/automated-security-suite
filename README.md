# Automated Security & Testing Suite
A Python-based DevSecOps framework that automates Security Scanning (SAST) and Unit Testing within a GitHub workflow.

## Features
- **Automated SAST:** Integrates Bandit for static analysis of Python code[cite: 34].
- **CI/CD Integration:** Automatically blocks "dirty" code from production via GitHub Actions[cite: 35].
- [**Code Coverage:** Ensures core API logic remains bug-free[cite: 35].
