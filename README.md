# Automated Security & Testing Suite
A Python-based DevSecOps framework that automates Security Scanning (SAST) and Unit Testing within a GitHub workflow.

## Features
- [cite_start]**Automated SAST:** Integrates Bandit for static analysis of Python code[cite: 34].
- [cite_start]**CI/CD Integration:** Automatically blocks "dirty" code from production via GitHub Actions[cite: 35].
- [cite_start]**Code Coverage:** Ensures core API logic remains bug-free[cite: 35].
